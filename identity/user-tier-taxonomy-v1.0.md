# User-Tier Taxonomy v1.0
## Tiers
| Tier | Name | Stability | Description | Examples |
| --- | --- | --- | --- | --- |
| 0 | Protocol Substrate | Static | Foundational identities anchored to protocol contracts and core registries. | Protocol Custodian, Root Key Steward, Chronos Root Agent |
 | 1 | Covenant Stewards & Governance Council | Anchored-Static | Identities empowered for policy, upgrades, and high-impact decisions. | Small Counsel, Counsel Signatory, Guardian-At-Large |
| 2 | System Architects & Core Agents | Anchored-Fluid | Designers and operators of major subsystems bridging protocol and users. | Vantage Architect, System Cartographer, Chronos-Audit Core Agent |
| 3 | Credentialed Operators & Chroniclers | Anchored-Fluid | Operational roles: curation, moderation, and narrative stewardship. | Alethea Historian, Chronos Field Reviewer, Ethos Curator |
| 4 | Registered Participants | Fluid (Anchored) | Everyday participants with registered avatars and basic verifions. | Citizen Avatar, Builder / Creator, Verifier-Participant |
| 5 | Ephemeral & Guest Identities | Ephemeral / Fluid | Session-bound or pseudonymous presences with minimal credentials. | Guest Explorer, Ephemeral Witness, Sandbox Avatar |

**Usage**: Guides sigil expression and provenance requirements across the Liquid Avatar ecosystem.
